## コミュニティ要望（本家Issue/Discussions由来）

### パフォーマンス関連
- WebSocket高負荷問題 → misskey-rsでは効率的なTokio実装
- リアクション処理の負荷 → カウンター直接更新で対応済み
- ActivityPub署名計算コスト → Ed25519移行検討

### 要望として上がっているが未対応
- Page機能の文字数上限緩和 → ✅ 実装済み（管理者設定可能）
- 代名詞（Pronouns）機能 → ✅ 実装済み
- 未添付ドライブデータ一括削除 → ✅ 実装済み
- デフォルトぼかし効果設定 → ✅ 実装済み
